# Python Task Manager

A simple command-line task manager built with Python.

This project allows users to manage daily tasks directly from the terminal.

## Features

- Add tasks
- View tasks
- Remove tasks
- Save tasks to a file
- Load tasks automatically when the program starts

## Technologies

- Python
- Command Line Interface (CLI)
- File handling

## How to Run

1. Install Python
2. Clone the repository

git clone https://github.com/jeanegdss/python-task-manager

3. Run the program

python task_manager_file.py
