# Python Task Manager

A simple command-line task manager built with Python.

## Features

- Add tasks
- View tasks
- Remove tasks
- Save tasks to file
- Load tasks automatically

## Technologies

- Python
- File handling
- Command Line Interface (CLI)

## How to run

1. Install Python
2. Download the project
3. Run:

python task_manager_file.py
